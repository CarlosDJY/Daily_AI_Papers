论文核心解决方案：监督强化学习 (SRL) 框架

根据提供的论文片段，本文提出的核心解决方案是一个名为监督强化学习（Supervised Reinforcement Learning, SRL）的训练框架。该框架旨在解决大型语言模型（LLMs），特别是小规模模型，在处理需要多步复杂推理的任务（如高级数学推理和软件工程）时遇到的学习困难。

SRL通过将复杂问题的解决过程重新定义为一个序列决策任务，巧妙地结合了监督学习的精细指导和强化学习的探索优化，从而克服了传统训练方法（如监督微调SFT和基于最终结果的强化学习RLVR）的局限性。

一、 SRL的核心思想与目标

SRL的核心思想是：与其让模型直接模仿最终的完整答案，不如教会它模仿专家的“思考过程”。它将一个复杂的解决方案分解为一系列逻辑连贯的“动作”（action steps），并通过在每一步提供密集的反馈信号来指导模型学习。

主要目标： 1. 提升复杂推理能力：使模型能够掌握交错规划、动态调整和自我验证等高级推理模式。 2. 解决学习信号稀疏问题：在传统RL中，模型只有在得出最终正确答案时才能获得正奖励，导致学习效率低下。SRL通过每一步的反馈提供密集的学习信号。 3. 增强模型的可解释性：通过生成内部推理独白，使模型的决策过程更加透明。 4. 赋能小型模型：让计算资源有限的小型开源模型也能学习并解决此前难以攻克的复杂问题。

二、 SRL框架的具体实施步骤

SRL的实现主要包含以下四个关键步骤：

1. 将专家解决方案分解为动作序列

首先，SRL将一个由专家（或强大的教师模型）生成的完整解决方案轨迹（trajectory）分解为一系列离散的、有逻辑意义的动作步骤。 - 在数学推理中，一个动作可以是执行一次代数运算或逻辑推导。 - 在软件工程中，一个动作可能是一个API调用、一个文件编辑命令或一段代码的编写。

2. 构建逐步式（Step-wise）训练数据

为了让模型学习如何在任意中间状态下做出正确的下一步决策，SRL将一个完整的专家轨迹转换成多个训练样本。对于一个包含 N 个步骤的专家轨迹，可以构造出 N-1 个训练实例： - 输入（上下文）：提供问题描述以及从第1步到第 k-1 步的专家动作。 - 目标（标签）：要求模型预测出第 k 步的专家动作。

通过这种方式，单个专家解决方案被转化为丰富的训练数据，教会模型如何从各种中间状态正确地推进。

3. 生成带有内部独白的动作

在训练和推理过程中，模型在每一步被提示先生成一段内部推理独白（internal reasoning monologue），通常被包裹在 <think> 标签内。这段独白帮助模型“思考”当前状态、规划下一步行动，然后再生成实际的动作。 - 模型输出格式：[<think>...</think>, action_step] - 这种设计不仅提升了模型的决策质量，也使其推理过程变得更加透明和可解释。

4. 采用平滑的、基于相似度的奖励机制

这是SRL与传统RL最显著的区别。在每个决策步骤，模型会获得一个“平滑”的奖励信号。这个奖励并非二元（对/错），而是根据模型生成的动作与专家动作之间的序列相似度来计算的。 - 奖励函数：R(model_action, expert_action) = SimilarityScore - 优势：即使模型的动作不完全正确，只要方向大体一致，它依然能获得部分奖励。这种密集的、细粒度的奖励为模型提供了持续的学习信号，极大地提高了训练效率，尤其是在模型探索的早期阶段。如果模型输出格式错误，则会受到负奖励。

三、 SRL的优势与应用

优势总结
  • 克服传统方法局限：有效解决了SFT的死板模仿和RLVR奖励稀疏的问题。
  • 促进高级推理：SRL能引导模型学习灵活的推理模式，如在解题过程中进行规划、执行、反思和验证的交错循环，从而提高解决方案的质量，而不仅仅是增加输出长度。
  • 提高训练效率：通过动态采样等技术，可以筛选掉学习信号较弱（即奖励方差接近于零）的样本，进一步优化训练过程。
  • 通用性强：该框架被证明在数学推理（如AMC、AIME等竞赛级问题）和代理软件工程（如根据需求自动生成代码补丁）等不同领域均表现出色。
训练与优化流程
  1. 初步训练：使用SRL框架对模型进行训练，使其掌握逐步推理和模仿专家行为的能力。
  2. 进一步微调：在SRL训练之后，可以继续使用RLVR（基于最终结果的强化学习）进行微调，以实现最佳性能。这种组合形成了一种强大的课程学习策略。

四、 结论

综上所述，监督强化学习（SRL）框架通过将复杂问题分解为可管理的步骤、引入内部推理独白，并采用基于相似度的密集奖励机制，为训练大型语言模型掌握高级推理技能提供了一条有效且高效的路径。它不仅显著提升了模型在数学和软件工程等挑战性任务上的性能，也为未来开发更强大、更具适应性的人工智能代理奠定了坚实的基础。
